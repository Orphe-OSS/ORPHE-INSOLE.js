# INSOLE MUSIC SHOE

ORPHE INSOLE を手に持って演奏する**ジェスチャ楽器** example。ミニマルなモノクロームの音世界（サイン波ブリップ・クリック・ノイズバースト・サブキック）と、音の波形が光として蓄積するビジュアライザを持ちます。音はすべて Web Audio で合成（サンプルファイル不使用）。

## 2つのページ

| ページ | 役割 |
|---|---|
| `index.html` | 楽器本体 |
| `lab.html` | **GESTURE LAB** — ジェスチャごとの生データをCSV記録。判定しきい値はこのデータから較正済み |

## インタラクション（手持ち）

| ジェスチャ | 判定 | 音 (L バンク / R バンク) |
|---|---|---|
| つま先を下にして振る/叩く | 重力LP向き + \|acc\| 窓ピーク | HAT（クリック / 高域サインブリップ） |
| 水平にして振る/叩く | 〃 | SNR（ノイズバースト / ホワイトバースト） |
| 踵を下にして振る/叩く | 〃 | KIK（808サブ / ハードサインパンチ） |
| 長軸まわりロール + ひねり | ジャイロ積分（窓120ms） | MOD+（サイン・グリッサンド上昇 / 階段状サインアルペジオ上昇） |
| 長軸まわりロール − ひねり | 〃 | MOD−（サイン急降下＋トレモロ / 階段状下降＋サブブーム） |
| FSR 各chを指で押す | ベースライン差分 Schmitt | ノート×6（L=サインパッド / R=サインリード+1oct、Aマイナーペンタ） |
| 水平のまま回す（ターンテーブル） | ヨー角速度（FLAT時のみ） | 正転=ループ再生（速度追従）/ 逆転=逆再生 / 往復=スクラッチ（共有プレイヘッド）。逆なら `TUNING.yawSign` 反転 |

- **L/R バンク**は `mount_position` から自動判定（不明時は id0=L, id1=R）
- ハードウェアなしでも**クリック / キーボード**で全音色を試奏可能（ページ下部の KEYS 参照、SPACE=ループREC）

## 演奏を支える仕組み

- **固定レイテンシスケジューラ**: BLE転送ジッタ（実測 p50 35ms / p95 86〜150ms）を、センサのデバイスタイムスタンプ基準のスケジューリングで吸収。ジェスチャ→発音の遅延が毎回一定になる。バジェットは実測ジッタに自動追従（`latencyAuto`）、tickerに JIT p95 / BUDGET を常時表示
- **QUANT**（デフォルトON）: 最初の一打を基準に16分グリッドへ吸着。生演奏がそのままグルーヴにロックされる
- **LOOP STATION**: REC→REC（ループ長確定）→REC（オーバーダブON/OFF）。ノートのON/OFF（押している長さ）も記録
- **LIGHT SCOPE**: 黒背景に波形が白い光の糸として加算合成で蓄積（音圧=明度、重なるほど明るい）。打撃で全面ストロボ＋バキバキの明滅、スクラッチ/ロールひねりでは波がうねる別エフェクト。すべて白黒
- **TUNE パネル**: 全パラメータをライブ編集（自動保存）。COPY JSON で調整値を共有可能

## 較正メモ

しきい値は GESTURE LAB の実収録CSV（`recordings/`、L/R両個体）から較正済み:
打撃=窓ピーク+再アーム式（誤爆ゼロ実証）、向き=遅い重力LP＋非対称しきい値、ひねり=戻しスイング抑制付き積分、FSR=`noteDelta` 200カウント。
ゾーン割当や軸が合わない個体は **CAL**（つま先下でクリック）と **ZERO** で較正。

## 実装メモ

- Streaming mode 3（200Hz, gyro+acc+press）。Lab は mode 4（100Hz, quat付き）で記録
- 描画は rAF スロットリング、ビジュアライザは `globalCompositeOperation: 'lighter'` による加算残光
- `AudioBufferSourceNode.playbackRate` は負値非対応のため、逆再生は反転バッファ＋共有プレイヘッドで実装
