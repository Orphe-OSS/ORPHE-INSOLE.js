Left / Right Ratio

緑線は全荷重のうち左足に乗っている割合です。破線の50%に近いほど左右均等で、上なら左足荷重、下なら右足荷重です。

Steadiness -- 100に近いほど、この試行内では揺れが少なく安定。
Path -- 重心点が動いた距離の合計。小さいほど揺れが少ない。
Velocity -- 1秒あたりの揺れ量。大きいほど速く補正している。
Ellipse -- 揺れが広がった範囲。大きいほど広く動いている。
L/R Load -- 左/右の荷重割合。50/50に近いほど左右均等。

How to Test

検査の進め方

  1. 安全のため、支えられる人が横に立ちます。足幅・靴・床・姿勢を毎回そろえます。
  2. 開眼: 正面の一点を見て、話さず静かに立ちます。Centerで現在位置を基準にしてStartします。
  3. 閉眼: 開眼で安全に立てる場合だけ実施します。目を閉じると視覚情報が減るため揺れやすくなります。
  4. 足が動く、支えが必要、強い不安がある場合は中止し、その試行は比較から外します。

Posture

姿勢

両足でまっすぐ立ち、腕は自然に下ろします。膝を固めすぎず、検査中は足を踏み替えません。

Result Guide

結果の見方

まずCoP軌跡の大きさを見ます。次に軌跡長・速度・楕円面積で揺れ量を確認し、最後に左右荷重の偏りを見ます。

Compare

比較のポイント

数値は単独で診断せず、同じ人・同じ条件で比較します。開眼より閉眼で大きく悪化する場合は、視覚に頼った姿勢制御の可能性を疑います。
